# maran-fill

Automation fill-in virtual bill in the [Ekuatia](https://ekuatia.set.gov.py/ekuatiai/) system.

The Ekuatia system is a a platform for contributers to generate and emit Electronic Tax Documents.

The Ekuatia’i System may only be used by taxpayers with a single establishment registered in the RUC (Single Taxpayer Registry) and a single point of issuance. Furthermore, access to the Ekuatia’i System is only permitted with the taxpayer's access code and cannot be granted through an authorized third party.
